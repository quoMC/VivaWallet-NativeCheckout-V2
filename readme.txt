== Description ==

 Native Checkout v2.0 from Viva Payments, allows merchants to accept payments natively on their ecommerce store. The card details are harvested natively on your site and are send to Viva Payments directly - without touching your server! SCA, PSD2 and 3D-Secure compatible.

 Features:
   • The payment begins and ends at your website. No redirects.
   • Supports 3D-Secure, SCA (PSD2 compliant)
   • PCI-DSS compliant
   • Plug-n-Play

› You will need your API v.2.0 credentials!
› WooCommerce Only
› The plugin does not support installments and any other currency other than your account's default.
› This project is maintained by an independent developer, not Viva Payments. It is based on their API documentation on Github [https://github.com/VivaPayments/API].

== Installation ==

Follow the guide at(https://developer.vivawallet.com) to create a demo or live account in Viva dashboard and then use the necessary credentials in the Woocommerce payment gateway options.
